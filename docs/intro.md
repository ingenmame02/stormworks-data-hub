# Stormworks Data Hubへようこそ

このサイトは、Stormworksの仕組みや作り方を少しずつ整理していくためのwikiです。

まずは「1ページずつ記事を増やす」ことを目標にします。Docusaurusでは、`docs` フォルダにMarkdownファイルを追加すると、その内容がWebページになります。

## 最初に覚えること

- 記事は `docs` フォルダに置きます。
- 見出しは `#` や `##` で書きます。
- 注意書きは `:::note` や `:::warning` のような記法で書けます。
- サイト全体の名前やメニューは `docusaurus.config.ts` で設定します。
