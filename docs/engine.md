# モジュラーエンジンの基本構造

Stormworksにおけるモジュラーエンジンは、複数のパーツを組み合わせて自作する高出力なエンジンです。

## 必須のコンポーネント

C++やLuaと同様に、エンジンを動かすにも適切な「入力」と「出力」が必要です。

* **Crankshaft（クランクシャフト）**: エンジンの芯となるパーツ。
* **Cylinder（シリンダー）**: 燃料を爆発させる場所。
* **Carburetor（キャブレター）**: 空気と燃料を混ぜる。

:::warning
冷却（Cooling）を怠ると、エンジン温度が115℃を超えたあたりで**大爆発**します。ポンプとラジエーターは必ず接続しましょう。
:::
